# Quantum Visualizer

Quantum Visualizer is a Python project that explores quantum mechanics through numerical computation and visualization.

Many quantum mechanics concepts are introduced through mathematical equations, which can make it difficult to build intuition for what a 
wavefunction, probability density, or quantum system actually looks like. This project aims to bridge that gap by using scientific computing 
tools to visualize and experiment with fundamental quantum mechanics concepts.

## Goals

* Visualize wavefunctions and probability densities
* Explore quantum systems through computation
* Build intuition for abstract quantum concepts
* Practice scientific computing with Python, NumPy, and Matplotlib

## Technologies

* Python
* NumPy
* Matplotlib
* Pytest

## Status

Currently in development.

Planned topics include:

* Wavefunction normalization
* Particle-in-a-box states
* Gaussian wave packets
* Quantum tunneling
* Additional visualizations that help make quantum mechanics more enjoyable
